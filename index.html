Bish-Bosh 2.0



A brief explaining the concept of the application

Loop value is the value range you want to operate on. The first division value will divide the current loop value, and if it is evenly divided it will display "Bish". The second divisionvalue will divide the current loop value, and if it is evenly divided it will display "Bosh". If both first and second value is evenly divided then it will show "Bish-Bosh". If none of the conditions are met, loop value number will be shown.
